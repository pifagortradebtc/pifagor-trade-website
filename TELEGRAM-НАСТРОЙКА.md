# Как открыть сайт Pifagor Trade через Telegram

Сайт pifagor.trade можно открывать как **Telegram Mini App** прямо в приложении.

---

## Способ 1: Кнопка меню (рекомендуется)

При нажатии на кнопку слева от поля ввода в чате с ботом откроется сайт.

### Шаги

1. Откройте **@BotFather** в Telegram
2. Отправьте команду **`/setmenubutton`**
3. Выберите вашего бота (того, с которым связан Pifagor Trade)
4. Когда спросят тип кнопки — выберите **Web App**
5. Введите **URL**: `https://www.pifagor.trade`
6. Введите текст кнопки, например: **Сайт** или **Открыть приложение**

Готово. Теперь при открытии чата с ботом пользователь увидит кнопку «Сайт», и при нажатии откроется pifagor.trade внутри Telegram.

---

## Способ 2: Через Bot Settings (то же, что способ 1)

1. Откройте **@BotFather**
2. Отправьте **`/mybots`**
3. Выберите нужного бота
4. **Bot Settings** → **Menu Button**
5. **Configure menu button**
6. URL: `https://www.pifagor.trade`
7. Текст кнопки: **Сайт** (или любой другой)

---

## Способ 3: Main Mini App (кнопка в профиле бота)

С Bot API 7.8+ можно сделать кнопку Mini App в профиле бота.

1. Откройте **@BotFather**
2. **`/mybots`** → выберите бота
3. **Bot Settings** → **Configure Mini App** → **Enable Mini App**
4. Укажите URL: `https://www.pifagor.trade`

---

## Способ 4: Прямая ссылка

Можно делиться ссылкой на Mini App:

```
https://t.me/ИМЯ_ВАШЕГО_БОТА?startapp
```

Например, если бот называется @PifagorTradeBot:

```
https://t.me/PifagorTradeBot?startapp
```

---

## Важно

- Сайт должен открываться по **HTTPS** (www.pifagor.trade уже настроен)
- Используйте **того же бота**, который указан в `TELEGRAM_BOT_TOKEN` (если у вас один бот для сигналов/приветствия) или создайте отдельного бота специально для сайта

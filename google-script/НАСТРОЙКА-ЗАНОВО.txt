НАСТРОЙКА ЗАНОВО — пошагово
================================

ШАГ 1. Новая таблица Google
---------------------------
1. Откройте: https://sheets.google.com
2. Нажмите «+» или «Пустая таблица»
3. Назовите таблицу (например: Рефералы Pifagor)
4. Скопируйте ID из адресной строки:
   https://docs.google.com/spreadsheets/d/ВОТ_ЭТОТ_ID/edit
   (только ID, без https://...)

ШАГ 2. Apps Script — привязать к этой таблице
----------------------------------------------
Вариант А (рекомендуется): скрипт ВНУТРИ таблицы
1. В вашей новой таблице: меню Расширения → Apps Script
2. Удалите весь код в редакторе
3. Скопируйте код из файла Code.gs в проекте
4. Сохраните (Ctrl+S)
5. Развернуть → Новое развёртывание
   - Тип: Веб-приложение
   - Выполнять от имени: я
   - У кого доступ: Все пользователи
   - Развернуть
6. Скопируйте URL (заканчивается на /exec)

Вариант Б: отдельный скрипт (script.google.com)
1. Откройте: https://script.google.com
2. Новый проект
3. Вставьте код из Code.gs
4. Файл → Свойства проекта → Script Properties
   Добавить: SPREADSHEET_ID = ID вашей таблицы из шага 1
5. Сохранить
6. Развернуть → Новое развёртывание
   - Веб-приложение, Все пользователи
7. Скопируйте URL

ШАГ 3. Обновить .env
--------------------
GOOGLE_SCRIPT_URL=ваш_новый_url

ШАГ 4. Проверка
----------------
npm run test-referral

Если «✓ Успех!» — откройте таблицу, обновите страницу.
Должен появиться лист «Рефералы» с тестовой строкой.

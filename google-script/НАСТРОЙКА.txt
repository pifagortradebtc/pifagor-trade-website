БЫСТРАЯ НАСТРОЙКА (5 минут)
============================

1. Откройте: https://sheets.google.com → Создать → Пустая таблица

2. Назовите таблицу (например: Подписчики Pifagor)

3. Лист «Путешествия»: в A1 впишите  email  (только одна колонка)

4. Меню: Расширения → Apps Script

5. Удалите весь код в редакторе

6. Откройте файл Code.gs из папки google-script в этом проекте
   Скопируйте ВЕСЬ код и вставьте в редактор Apps Script

7. Сохранить (Ctrl+S)

8. Развернуть → Новое развёртывание
   - Тип: Веб-приложение
   - Выполнять от имени: я
   - У кого доступ: Все пользователи (важно!)
   - Развернуть

9. Разрешите доступ (если попросит)

10. Скопируйте URL (должен заканчиваться на /exec)

11. Добавьте в .env проекта:
    GOOGLE_SCRIPT_URL=https://script.google.com/macros/s/ВАШ_ID/exec
    GOOGLE_SHEET_URL=https://docs.google.com/spreadsheets/d/ID_ТАБЛИЦЫ/edit (опционально)

12. Если скрипт создан НЕ из таблицы (standalone): Файл → Свойства проекта →
    Script Properties → добавить SPREADSHEET_ID = ID таблицы из URL

13. Готово! Данные будут появляться в листах «Путешествия» (email) и «Рефералы».

РЕФЕРАЛЫ НЕ ПОЯВЛЯЮТСЯ В ТАБЛИЦЕ?
=================================
- Проверьте GOOGLE_SCRIPT_URL в .env (или Render Environment)
- URL должен заканчиваться на /exec
- Развёртывание: «У кого доступ: Все пользователи»
- Если скрипт standalone: задайте SPREADSHEET_ID в Script Properties
- Лист «Рефералы» создаётся автоматически при первой записи
